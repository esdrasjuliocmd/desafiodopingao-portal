# desafiodopingao-portal

Portal oficial de jogos para o domínio [desafiodopingao.com.br](https://desafiodopingao.com.br).

## Sobre

Este repositório contém a base do portal **Desafio do Pingão**. Todos os jogos são acessados a partir de botões neste portal.

## Estrutura

```
├── index.html   # Página principal do portal
├── style.css    # Estilos do portal
└── README.md    # Este arquivo
```

## Como usar

Basta abrir o arquivo `index.html` em qualquer navegador, ou hospedar os arquivos no servidor do domínio `desafiodopingao.com.br`.

## Adicionando novos jogos

Para adicionar um novo jogo, insira um novo `<a class="game-card">` dentro da `<div class="games-grid">` em `index.html`, seguindo o padrão dos cards existentes.
